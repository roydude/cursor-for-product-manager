# PRD (Product Requirements Document)

## 개요
제품 요구사항 문서(PRD)를 관리하는 폴더입니다.

## PRD 작성 가이드
1. **제목**: [기능명] PRD_[작성일자]
2. **구조**: 개요 → 타깃 사용자 → 사용자 시나리오 → 핵심 기능 → 성공 지표 → 비회원 정책 → 활용 데이터 → 부록
3. **버전 관리**: 문서 하단에 버전 정보 및 승인자 정보 포함
